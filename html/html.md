### HTML ? 
```
Htper Text Markup Language
- 웹 페이지 표시를 위해 콘테치를 삽일할 수 있게 해주는 요소들로 구성된 간단한 언어
```

### HTML 구성요소
```
* 태그
 - 웹 문서를 구성하는 명령어

* 요소
 - 빈 요소 : br, hr
 - 블록 요소 : h1~h6, div, list, p 등
 - 인라인 요소 : a, span strong
```

### HTML 기본 태그와 역할
```
 - Heading : 제목 사용
 - p : 문단 정의
 - pre : 미리 정의된 형식의 텍스트를 정의할 때 사용
 - ol : 순서가 있는 목록 (1,2,3... , a,b,c, ...)
 - ul : 순서가 없는 목록 (*)
 - dl : 설명 목록 (-)
 - br : 줄바꿈
 - hr : 수평선
 - 텍스트 서식 : strong, b, em, i, small, mark, sub, sup
 - 인용 : blockquote, q
```
